موقع الحقائب التدريبية — جاهز لـ GitHub Pages

محتويات المجلد:
1) index.html
   الصفحة الرئيسية، وتحتوي على مربعين لاختيار الحقيبة.

2) bag.html
   الصفحة الفرعية الموحدة لعرض كل ملف PDF بحسب الاختيار من index.html.

3) ملفا PDF بالأسماء الأصلية:
   - حقيبة أساسيات نظام الشركات.pdf
   - حقيبة ايداع القوائم المالية تحت التعديل.pdf

4) صورتا الغلاف:
   - companies-cover.jpg
   - financial-cover.jpg

طريقة الرفع:
- ارفع جميع الملفات كما هي داخل جذر مستودع GitHub.
- من Settings > Pages اختر Deploy from a branch ثم main / root.
- لا تغيّر أسماء ملفات PDF إلا بعد تحديثها داخل bag.html.
