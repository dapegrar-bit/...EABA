طريقة الرفع على GitHub Pages:
1) فك الضغط عن الملف.
2) ارفع جميع الملفات الموجودة داخله إلى المجلد الرئيسي للمستودع.
3) لا ترفع الملف المضغوط نفسه.
4) تأكد من وجود index.html و bag.html وملفات الصور وPDF في المستوى نفسه.

مهم:
- الصفحة الرئيسية: index.html
- صفحة العرض المشتركة: bag.html
- استُخدمت أسماء إنجليزية داخلية للصور وملفات العرض لتجنب مشكلة الأحرف العربية المركبة في روابط GitHub Pages.
- بقيت نسخ ملفات PDF بالأسماء العربية المطلوبة داخل المشروع.
